# 测试资源

本目录存放 DefaultHound 的测试用资源文件。

## 文件说明

| 文件 | 说明 |
|---|---|
| `targets.txt` | 目标列表，每行一个目标，用于扫描测试 |

## targets.txt 格式

支持以下格式：

| 格式 | 示例 | 说明 |
|---|---|---|
| `ip` | `172.20.0.2` | 扫描所有服务（各自默认端口） |
| `ip:port` | `172.20.0.2:3306` | 仅扫描匹配该端口的服务 |
| `ip:port1,port2` | `172.20.0.2:3306,6379` | 扫描匹配这些端口的服务 |
| `ip:service^port` | `172.20.0.2:druid^8888` | 指定服务使用自定义端口 |
| `ip:service^` | `172.20.0.2:redis^` | 指定服务使用默认端口 |
| `# 注释` | `# MySQL` | 注释行，会被忽略 |
